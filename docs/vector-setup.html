Steps

導入手順

1

Vector.cls をダウンロードする

Vector は単体で使えるクラスです。まずは下のリンクから Vector.cls を保存します。

Vector.clsそのままインポートできるクラスモジュール
2

Excel の VBE を開く

対象ブックを開いた状態で Alt + F11 を押し、Visual Basic Editor を開きます。Project Explorer が見えていない場合は Ctrl + R で表示します。

3

クラスモジュールをインポートする

VBE のメニューから ファイル > ファイルのインポート を選び、保存した Vector.cls を指定します。Project Explorer の Class Modules 配下に Vector が追加されれば取り込み完了です。

4

正常に取り込めたか確認する

Project Explorer に Vector が見えていることを確認したうえで、イミディエイトウィンドウを開いて型名を確認します。

? TypeName(New Vector)

Checks

インポート確認

  • Project Explorer の Class Modules に Vector が追加されていること
  • イミディエイトウィンドウで TypeName(New Vector) を実行し、Vector と表示されること
  • この段階ではまだサンプルデータの読み込みや集計を行わず、取り込み確認だけで止めること
