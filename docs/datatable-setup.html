DataTable

  • 導入方法
  • 使い方
  • リファレンス

Steps

導入手順

1

必要な cls をまとめてダウンロードする

DataTable は公開シグネチャの中で Matrix と Vector を参照します。そのため、次の 3 ファイルをそろえて保存します。

DataTable.cls列名付き表を扱う本体クラス Matrix.cls内部で 2 次元データを扱うために必要 Vector.cls列を Vector として返す API で必要
2

Excel の VBE を開く

対象ブックを開いた状態で Alt + F11 を押し、Visual Basic Editor を開きます。Project Explorer が見えていない場合は Ctrl + R で表示します。

3

3 つのクラスモジュールをインポートする

VBE のメニューから ファイル > ファイルのインポート を選び、Vector.cls、Matrix.cls、DataTable.cls の順で取り込みます。Project Explorer の Class Modules に 3 つが並べば準備完了です。

4

正常に取り込めたか確認する

依存クラスまで正しく入っていれば、イミディエイトウィンドウから DataTable をそのまま生成できます。

? TypeName(New DataTable)

Checks

インポート確認

  • Project Explorer の Class Modules に Vector、Matrix、DataTable が並んでいること
  • イミディエイトウィンドウで TypeName(New DataTable) を実行し、DataTable と表示されること
  • このページではまだヘッダー付き表の読込や抽出には進まず、インポート確認だけで止めること
