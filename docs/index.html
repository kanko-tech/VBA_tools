Structure

ドキュメントの見方

Guide

README の内容を拡張したページです。用途、向いている場面、導入方法、最小例を整理してあります。

Reference

プロパティとメソッドの全解説ページです。シグネチャ、役割、注意点を確認しながら実装できます。

Modules

モジュール一覧

Vector

1 列または 1 行の値を読み込み、条件マスク、型変換、集計に使うクラスです。

Guide Reference

Matrix

列名を持たない表データを 2 次元配列として扱うための基本クラスです。

Guide Reference

DataTable

ヘッダー付きの表データを列名ベースで扱うためのクラスです。

Guide Reference

Choose

使い分けの目安

Vector を選ぶ場面

列や行の値だけをまとめて扱いたいとき。条件判定や集計を独立した部品として使いたいとき。

Matrix を選ぶ場面

列名は不要で、表全体を配列として組み替えたいとき。中間処理の土台に向いています。

DataTable を選ぶ場面

列名を基準に抽出や更新を書きたいとき。実務データを読みやすく保守したいときに向いています。

Quick Start

導入の流れ

1. Overview を確認

どのモジュールを使うべきかを先に確認します。

2. Guide を読む

用途、導入方法、最小例を確認して、使い始め方をつかみます。

3. Reference を参照

実装時に必要なプロパティやメソッドを詳細ページで確認します。

4. README とソースも確認

実装寄りの補足は各 README とクラス本体をあわせて見ます。
