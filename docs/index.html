How To Read

ページの役割

Guide

README を拡張した案内ページです。用途、向いている場面、導入方法、最小例を先に把握できます。

Reference

プロパティとメソッドの技術詳細です。クラス図、前提条件、注意点、ユースケースを見ながら実装できます。

Modules

モジュール一覧

Vector

一次元データを読み込み、条件マスク、型変換、集計、出力までを扱うクラスです。

Guide Reference

Matrix

列名を持たない表データを、二次元配列として組み替えたり抽出したりするための基盤クラスです。

Guide Reference

DataTable

ヘッダー付き表データを列名ベースで扱い、実務寄りの抽出や更新を読みやすく書くためのクラスです。

Guide Reference

Choose

使い分けの目安

Vector を選ぶ

列や行の値だけをまとめて扱い、条件判定や集計を独立した部品として使いたいときに向いています。

Matrix を選ぶ

列名を使わず、表全体を配列として切り出したり並べ替えたりする中間処理の土台に向いています。

DataTable を選ぶ

列名を基準に抽出や更新を書きたいときや、セル番地依存のコードを減らしたいときに向いています。

Flow

読み進め方

1. Home で選ぶ

まずどのモジュールを使うかを決めます。

2. Guide でつかむ

用途、導入方法、最小例を見て、使い始め方を把握します。

3. Reference で詰める

API の詳細、前提条件、注意事項、ユースケースを確認します。

4. README とソースで補う

実装に近い補足や内部動作は README とクラス本体をあわせて確認します。
