DataTable

ヘッダー付きの表を列名ベースで扱うためのクラスモジュール

  • 導入方法
  • 使い方
  • リファレンス

Properties

プロパティ

名前 型 説明
is_loaded Boolean ヘッダー付き表データが読み込まれているとき True を返します。
row_count Long データ行数を返します。ヘッダー行は含みません。
col_count Long 列数を返します。
column_names Variant 列名を 1 次元配列で返します。
matrix Matrix 内部データ本体を保持する Matrix を返します。

Methods

メソッド

カテゴリ シグネチャ 戻り値 説明
読込 read_range(rng As Range[, hasHeader]) Sub Range から表を読み込みます。既定では先頭行をヘッダーとして扱います。
読込 read_matrix(src As Matrix, columnNames As Variant) Sub 既存の Matrix と列名配列から DataTable を組み立てます。
列参照 col(columnName As String) Variant 指定列を 1 次元配列で返します。
列参照 col_vector(columnName As String) Vector 指定列を Vector として返します。
抽出 filter_by_mask(mask As Variant) DataTable 真偽値マスクに合う行だけを持つ新しい DataTable を返します。
抽出 filter_by_equals(columnName As String, matchValue As Variant) DataTable 指定列が指定値と等しい行だけを返します。
抽出 filter_by_in(columnName As String, matchValues As Variant) DataTable 指定列が候補配列のいずれかに一致する行だけを返します。
抽出 filter_by_contains(columnName As String, searchText As String[, caseSensitive]) DataTable 指定列に検索文字列を含む行だけを返します。
抽出 filter_by_all_equals(columnNames As Variant, matchValues As Variant) DataTable 複数列がすべて一致する行だけを返します。
抽出 filter_by_any_equals(columnNames As Variant, matchValues As Variant) DataTable 複数列のうちいずれかが一致する行だけを返します。
抽出 select_columns(columnNames As Variant) DataTable 指定した列名だけを残した新しい DataTable を返します。
更新 add_column(columnName As String[, values]) Sub 指定名の列を追加します。値を省略すると空の列を追加します。
更新 rename_column(oldName As String, newName As String) Sub 既存の列名を変更します。
更新 sort_by(columnName As String[, ascending]) Sub 指定列を基準に表全体を並び替えます。
更新 set_by_mask(mask As Variant, columnName As String, newValue As Variant) Sub マスクに一致する行だけ、指定列の値をまとめて更新します。
更新 set_by_equals(conditionColumnName As String, matchValue As Variant, targetColumnName As String, newValue As Variant) Sub 条件列が指定値と等しい行だけ、別の列を更新します。
更新 set_column(columnName As String, values As Variant) Sub 指定列を配列値で丸ごと置き換えます。
出力 to_range(topLeft As Range[, includeHeader]) Sub ヘッダー行の有無を選んで、指定セルから表を書き戻します。
状態管理 clear() Sub 列名と内部データを初期化します。
