Example 1

売上金額列を整形して合計と平均を出す

Dim amountVec As New Vector

With Worksheets("売上明細")
    amountVec.read_col_range .Range("F5:F12")
End With

amountVec.fill_empty 0
amountVec.cast_to_double_safe emptyAsZero:=True, invalidAsZero:=True

Debug.Print amountVec.sum
Debug.Print amountVec.mean

売上金額は 売上明細!F5:F12 を使います。空欄を 0 にそろえてから集計することで、月次の合計や平均をそのまま確認できます。

Example 2

10 万円超の案件マスクを作る

Dim amountVec As New Vector
Dim highAmountMask As Variant

With Worksheets("売上明細")
    amountVec.read_col_range .Range("F5:F12")
End With

amountVec.fill_empty 0
amountVec.cast_to_double_safe emptyAsZero:=True, invalidAsZero:=True
highAmountMask = amountVec.gt(100000)

作成した highAmountMask は、あとで Matrix.filter_rows や DataTable.set_by_mask へ渡せます。実務では「要確認金額以上の案件だけ拾う」用途に向きます。

Example 3

入金予定日列を日付型にそろえて別領域へ出す

Dim dueVec As New Vector

With Worksheets("売上明細")
    dueVec.read_col_range .Range("H5:H12")
End With

dueVec.cast_to_date_safe invalidAsEmpty:=True
dueVec.to_range_vertical Worksheets("売上明細").Range("J5")

未設定の入金予定日は空のまま残し、日付として読める値だけをそろえて J5 から確認できます。元の一覧は壊さず、横に確認列を作る想定です。
