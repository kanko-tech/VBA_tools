Matrix

Steps

導入手順

1

Matrix.cls をダウンロードする

Matrix は単体で使えるクラスです。まずは下のリンクから Matrix.cls を保存します。

Matrix.cls矩形データを扱うクラスモジュール
2

Excel の VBE を開く

対象ブックを開いた状態で Alt + F11 を押し、Visual Basic Editor を開きます。Project Explorer が見えていない場合は Ctrl + R で表示します。

3

クラスモジュールをインポートする

VBE のメニューから ファイル > ファイルのインポート を選び、保存した Matrix.cls を指定します。Project Explorer の Class Modules 配下に Matrix が追加されれば取り込み完了です。

4

正常に取り込めたか確認する

Project Explorer に Matrix が見えていることを確認したうえで、イミディエイトウィンドウから型名を確認します。

? TypeName(New Matrix)

Checks

インポート確認

  • Project Explorer の Class Modules に Matrix が追加されていること
  • イミディエイトウィンドウで TypeName(New Matrix) を実行し、Matrix と表示されること
  • このページではまだデータ抽出や列選択までは進めず、取り込み確認だけで止めること
