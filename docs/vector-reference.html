CLASS MODULE

Vector

1 次元データの読み込み、条件マスク、型変換、集計のためのクラスモジュール

  • 導入方法
  • 使い方
  • リファレンス

Properties

プロパティ

名前 型 説明
is_loaded Boolean 内部にデータが読み込まれているとき True を返します。
data Variant 内部で保持している 1 次元配列を返します。
count Long 読み込み済み要素数を返します。
item(index) Variant 指定した位置の要素を返します。添字は 1 始まりです。

Methods

メソッド

カテゴリ シグネチャ 戻り値 説明
読込 read_col_range(rng As Range) Sub 縦方向の Range から 1 列分の値を読み込みます。
読込 read_row_range(rng As Range) Sub 横方向の Range から 1 行分の値を読み込みます。
読込 read_array(arr As Variant) Sub 1 次元配列をそのまま内部データとして読み込みます。
確認 type_names() Variant 各要素の型名を 1 次元配列で返します。
条件判定 eq(matchValue As Variant) Variant 指定値と等しい要素を True にしたマスクを返します。
条件判定 ne(matchValue As Variant) Variant 指定値と等しくない要素を True にしたマスクを返します。
条件判定 gt(threshold As Variant) Variant 指定値より大きい要素を True にしたマスクを返します。
条件判定 ge(threshold As Variant) Variant 指定値以上の要素を True にしたマスクを返します。
条件判定 lt(threshold As Variant) Variant 指定値より小さい要素を True にしたマスクを返します。
条件判定 le(threshold As Variant) Variant 指定値以下の要素を True にしたマスクを返します。
条件判定 is_empty([treatBlankStringAsEmpty]) Variant 空値判定のマスクを返します。空文字を空として扱うかは引数で切り替えます。
型変換 cast_to_double_safe([emptyAsZero], [invalidAsZero], [treatDateAsInvalid]) Sub 内部データを安全に数値へ寄せます。変換失敗時の扱いは引数で制御します。
型変換 cast_to_string_safe([emptyAsBlank], [errorAsBlank]) Sub 内部データを文字列へ寄せます。空値やエラー値の扱いを調整できます。
型変換 cast_to_date_safe([invalidAsEmpty]) Sub 内部データを日付へ寄せます。日付化できない値は空にできます。
整形 fill_empty(fillValue As Variant[, treatBlankStringAsEmpty]) Sub 空値と判定された要素を指定値で埋めます。
整形 map(functionName As String) Sub 指定した関数名を各要素へ順番に適用して内部データを書き換えます。
集計 unique() Variant 重複を除いた値の配列を返します。
集計 sum() Double 数値要素の合計を返します。
集計 mean() Double 数値要素の平均を返します。
出力 to_range_vertical(topLeft As Range) Sub 内部データを縦 1 列としてシートへ書き戻します。
出力 to_range_horizontal(topLeft As Range) Sub 内部データを横 1 行としてシートへ書き戻します。
状態管理 clear() Sub 内部データと読み込み状態を初期化します。
