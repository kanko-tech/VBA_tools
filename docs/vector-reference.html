Properties

プロパティと参照系

Property is_loaded Property Get is_loaded() As Boolean 読込済みかを返す

値が読み込まれているかだけを返す軽量な確認用プロパティです。

入力なし
出力Boolean
前提なし
注意データ本体の妥当性までは判定しません。
ユースケース初回読込前に処理を止めるガードを書く。
Property data Property Get data() As Variant 内部の一次元配列を返す

内部データを複製して返します。外部の処理へ生の配列として渡したいときに使います。

入力なし
出力Variant の一次元配列
前提事前に読込済みであること。
注意返されるのは複製なので、編集しても元データは変わりません。
ユースケース自前の VBA ループ処理へ配列をそのまま渡す。
Property count Property Get count() As Long 要素数を返す

保持している要素数を返します。ループや空判定の基準になります。

入力なし
出力Long
前提事前に読込済みであること。
注意1 始まり配列かどうかに関わらず件数だけを返します。
ユースケース結果列を書き出す前に長さを確認する。
Property item Property Get item(ByVal index As Long) As Variant 指定位置の値を返す

インデックスを指定して個別の値を参照します。

入力index As Long
出力指定位置の値
前提読込済みで、index が配列範囲内であること。
注意範囲外なら例外になります。
ユースケース先頭値や特定行の値だけを確認したいとき。
Method type_names Function type_names() As Variant 各要素の型名を返す

各要素を TypeName 相当の文字列配列にして返し、データ品質の確認に使えます。

入力なし
出力型名を並べた一次元配列
前提事前に読込済みであること。
注意Error と Empty は特別扱いされます。
ユースケース数値列のつもりが文字列混在になっていないか確認する。

Loading

読込系

Method read_col_range Sub read_col_range(ByVal rng As Range) 1 列の Range を読む

縦方向の 1 列範囲を読み込み、一次元配列として保持します。

入力rng As Range
出力なし。内部状態を更新。
前提rng が Nothing ではなく 1 列であること。
注意複数列を渡すと例外になります。
ユースケースシートの金額列だけを集計前に読み込む。
Method read_row_range Sub read_row_range(ByVal rng As Range) 1 行の Range を読む

横方向の 1 行範囲を読み込みます。見出し行や横並びの設定値向けです。

入力rng As Range
出力なし。内部状態を更新。
前提rng が 1 行であること。
注意複数行は受け付けません。
ユースケース見出し行を読み込んで列順を調べる。
Method read_array Sub read_array(ByVal arr As Variant) 一次元配列を読む

既存の配列ロジックから Vector に載せ替える入口です。

入力arr As Variant の一次元配列
出力なし。内部状態を更新。
前提arr が一次元配列であること。
注意二次元配列は例外になります。
ユースケース他処理で作った配列に対して eq や sum を使う。

Conditions

条件マスク系

Methodeq Function eq(ByVal matchValue As Variant) As Variant一致判定のマスク

各要素が指定値と一致するかをブール配列で返します。

入力matchValue
出力Boolean の一次元配列
前提読込済みであること。
注意Error や Null は一致しません。
ユースケース特定コードに一致する行だけを別処理へ渡す。
Methodne Function ne(ByVal matchValue As Variant) As Variant不一致判定のマスク

指定値と一致しない要素を True にしたマスクです。

入力matchValue
出力Boolean の一次元配列
前提読込済みであること。
注意比較不能な値は False 扱いです。
ユースケース空欄以外、特定区分以外を抽出したいとき。
Methodgt Function gt(ByVal threshold As Variant) As Variantより大きい条件

しきい値より大きい要素に対するマスクを返します。

入力threshold
出力Boolean の一次元配列
前提比較可能な値が入っていること。
注意文字列など比較失敗時は False になります。
ユースケース売上が一定額を超えるデータだけ拾う。
Methodge Function ge(ByVal threshold As Variant) As Variant以上条件のマスク

しきい値以上の要素に対するマスクです。

入力threshold
出力Boolean の一次元配列
前提読込済みであること。
注意比較失敗時は False です。
ユースケース締切日以降のデータをまとめる。
Methodlt Function lt(ByVal threshold As Variant) As Variantより小さい条件

しきい値より小さい要素を判定します。

入力threshold
出力Boolean の一次元配列
前提比較可能な値が入っていること。
注意数値列向けの利用が安全です。
ユースケース閾値未満のデータを異常値候補として拾う。
Methodle Function le(ByVal threshold As Variant) As Variant以下条件のマスク

しきい値以下の要素を判定します。

入力threshold
出力Boolean の一次元配列
前提読込済みであること。
注意Null や Error は対象外です。
ユースケース期日以前の行を抜き出す条件を作る。
Methodis_empty Function is_empty(Optional ByVal treatBlankStringAsEmpty As Boolean = True) As Variant空欄判定のマスク

Empty と、必要なら空文字も空欄として判定します。

入力treatBlankStringAsEmpty As Boolean
出力Boolean の一次元配列
前提読込済みであること。
注意空文字を空欄扱いしない設定にもできます。
ユースケース未入力セルの補完対象を一括で見つける。

Transform

変換・更新系

Methodcast_to_double_safe Sub cast_to_double_safe(Optional ByVal emptyAsZero As Boolean = False, Optional ByVal invalidAsZero As Boolean = False, Optional ByVal treatDateAsInvalid As Boolean = True)安全に Double へ変換

空欄や不正値の扱いを指定しながら、要素を Double 中心へそろえます。

入力空欄・不正値・日付値の扱いを指定する 3 つのオプション
出力なし。内部状態を更新。
前提読込済みであること。
注意日付も数値化したい場合は treatDateAsInvalid:=False を検討します。
ユースケースCSV 由来の数値列を集計前に正規化する。
Methodcast_to_string_safe Sub cast_to_string_safe(Optional ByVal emptyAsBlank As Boolean = True, Optional ByVal errorAsBlank As Boolean = True)安全に文字列へ変換

空欄やエラー値の扱いを指定しながら文字列へそろえます。

入力emptyAsBlank, errorAsBlank
出力なし。内部状態を更新。
前提読込済みであること。
注意errorAsBlank:=False にするとエラー値混在時に例外になります。
ユースケース比較前に全要素を文字列化して表記揺れを吸収する。
Methodcast_to_date_safe Sub cast_to_date_safe(Optional ByVal invalidAsEmpty As Boolean = True)安全に Date へ変換

日付解釈できる値だけを Date に変換します。

入力invalidAsEmpty As Boolean
出力なし。内部状態を更新。
前提読込済みであること。
注意変換できない値を残したい場合はこのメソッドだけでは対応しません。
ユースケース日付列の比較前に型をそろえる。
Methodfill_empty Sub fill_empty(ByVal fillValue As Variant, Optional ByVal treatBlankStringAsEmpty As Boolean = True)空欄を埋める

空欄や空文字を指定値で埋めます。

入力fillValue, treatBlankStringAsEmpty
出力なし。内部状態を更新。
前提読込済みであること。
注意空欄判定の基準はオプションで変わります。
ユースケース未入力の数値欄を 0 で埋めてから集計する。
Methodmap Sub map(ByVal functionName As String)各要素へ関数を適用

Application.Run で各要素に同じ関数を適用し、値を更新します。

入力functionName As String
出力なし。内部状態を更新。
前提対象関数が引数 (value, index) を受け取れること。
注意関数が見つからない、または実行失敗すると例外になります。
ユースケース各値へ一律の丸めや文字列整形をかける。

Aggregate

集計系

Methodunique Function unique() As Variant重複を除いた値を返す

順序を保ったまま重複を除外した一次元配列を返します。

入力なし
出力重複なしの一次元配列
前提読込済みであること。
注意比較キーは内部で文字列化されます。
ユースケースカテゴリ一覧や候補リストを作る。
Methodsum Function sum() As Double合計を返す

数値要素の合計を返します。

入力なし
出力Double
前提全要素が厳密に数値型であること。
注意文字列の数値はそのままでは集計できません。必要なら先に変換します。
ユースケース金額列の総計を求める。
Methodmean Function mean() As Double平均を返す

sum / count に基づいて平均値を返します。

入力なし
出力Double
前提全要素が数値型であり、件数が 0 でないこと。
注意空欄混在のままでは想定外になりやすいです。
ユースケース評価点列の平均を確認する。

Output

出力系

Methodto_range_vertical Sub to_range_vertical(ByVal topLeft As Range)縦方向へ書き出す

ベクトルを 1 列としてシートへ出力します。

入力topLeft As Range
出力なし。シートへ書込。
前提読込済みで、topLeft が Nothing ではないこと。
注意既存セルの内容は上書きされます。
ユースケース抽出済みのコード一覧を別列へ出力する。
Methodto_range_horizontal Sub to_range_horizontal(ByVal topLeft As Range)横方向へ書き出す

ベクトルを 1 行としてシートへ出力します。

入力topLeft As Range
出力なし。シートへ書込。
前提読込済みであること。
注意横に広がるため、既存の見出し行と衝突しない位置を選びます。
ユースケースユニーク値一覧を見出し行のように並べる。

State

状態管理系

Method clear Sub clear() 内部状態を初期化する

同じインスタンスを再利用したいときに、中身を未読込状態へ戻します。

入力なし
出力なし。内部状態を初期化。
前提なし
注意保持していた値は失われます。
ユースケースループの中で同じ変数を使い回す前に状態を消す。
