Matrix

二次元表データモジュールの API 詳細

Matrix の公開 API を、表データの読込・抽出・追加・変形・出力という流れでまとめています。 各項目は折りたたみ式で、入力・出力・前提・注意・ユースケースをすぐ確認できます。

主な出力 Matrix / Variant 配列 / シート出力

Properties

プロパティと参照系

Propertyis_loaded Property Get is_loaded() As Boolean読込済みかを返す

内部状態が初期化済みかを返します。

入力なし
出力Boolean
前提なし
注意データ内容の正しさまでは保証しません。
ユースケース処理前に未読込のまま呼んでいないか確認する。
Propertyrow_count Property Get row_count() As Long行数を返す

現在の行数を返します。

入力なし
出力Long
前提読込済みであること。
注意空 Matrix は 0 を返します。
ユースケース抽出後の件数確認や出力範囲計算。
Propertycol_count Property Get col_count() As Long列数を返す

現在の列数を返します。

入力なし
出力Long
前提読込済みであること。
注意列選択後の構造確認に向いています。
ユースケース追加列の前後で構造が変わったか確認する。
Propertydata Property Get data() As Variant内部の二次元配列を返す

内部データを複製して返します。

入力なし
出力Variant の二次元配列
前提読込済みであること。
注意返り値の編集は元データへ反映されません。
ユースケースVBA の通常ループや他 API に配列を渡す。
Propertyitem Property Get item(ByVal rowIndex As Long, ByVal colIndex As Long) As Variant指定セルの値を返す

行番号と列番号で個別要素を参照します。

入力rowIndex, colIndex
出力指定位置の値
前提読込済みで、両インデックスが範囲内であること。
注意範囲外は例外になります。
ユースケース先頭セルや特定行列の値だけを確認する。
Methodrow_values Function row_values(ByVal rowIndex As Long) As Variant行を一次元配列で返す

指定行だけを一次元配列として返します。

入力rowIndex As Long
出力行データの一次元配列
前提読込済みで行番号が有効であること。
注意列数ぶんの配列が返ります。
ユースケース特定レコードだけ別処理へ渡す。
Methodcol_values Function col_values(ByVal colIndex As Long) As Variant列を一次元配列で返す

指定列を一次元配列として返します。

入力colIndex As Long
出力列データの一次元配列
前提読込済みで、行数が 1 件以上あること。
注意行数 0 の Matrix では返せず例外になります。
ユースケース列単位の判定を Vector に渡す。

Loading

読込系

Methodread_range Sub read_range(ByVal rng As Range)矩形 Range を読む

シート上の矩形範囲を二次元配列として読み込みます。

入力rng As Range
出力なし。内部状態を更新。
前提rng が Nothing ではないこと。
注意単一セルでも 1x1 の Matrix として扱われます。
ユースケースシート上の中間表を丸ごと読み込む。
Methodread_array Sub read_array(ByVal arr As Variant)二次元配列を読む

既存の二次元配列をそのまま Matrix に載せます。

入力arr As Variant の二次元配列
出力なし。内部状態を更新。
前提arr が二次元配列であること。
注意三次元以上は例外になります。
ユースケースAPI や他クラスが返した配列を Matrix 化する。
Methodread_empty Sub read_empty(ByVal colCount As Long)空の Matrix を作る

列数だけを持つ、行数 0 の空 Matrix を初期化します。

入力colCount As Long
出力なし。内部状態を更新。
前提colCount >= 1
注意空 Matrix は一部の参照系メソッドで制約があります。
ユースケース抽出結果が 0 件でも列構造を保ちたいとき。

Selection

抽出・選択系

Methodfilter_rows Function filter_rows(ByVal mask As Variant) As Matrixマスクで行抽出

ブール配列に応じて行だけを抽出した新しい Matrix を返します。

入力mask の一次元配列
出力Matrix
前提mask の長さが行数と一致すること。
注意1 件も残らない場合は空 Matrix が返ります。
ユースケースVector で作った条件マスクを使って表全体を絞る。
Methodslice_rows Function slice_rows(ByVal rowIndexes As Variant) As Matrix指定順で行を切り出す

行番号配列に従って、必要な行だけを順番どおりに取り出します。

入力rowIndexes の一次元配列
出力Matrix
前提各行番号が有効範囲であること。
注意重複番号を指定すると同じ行が重複して入ります。
ユースケースソート済みインデックスに従って行を並び替える前処理。
Methodselect_columns Function select_columns(ByVal columnIndexes As Variant) As Matrix指定列だけ残す

必要な列番号だけを残して新しい Matrix を返します。

入力columnIndexes の一次元配列
出力Matrix
前提1 件以上の列番号があり、各番号が有効であること。
注意空配列は例外になります。
ユースケース中間表から必要列だけを抜き出して別処理に渡す。

Mutation

更新・追加系

Methodset_column_values Sub set_column_values(ByVal colIndex As Long, ByVal values As Variant)列全体を置換する

指定列を新しい値配列で一括置換します。

入力colIndex, values の一次元配列
出力なし。内部状態を更新。
前提values の長さが行数と一致すること。
注意行数 0 の Matrix では何も起きません。
ユースケース計算済みの新列値を既存列へ差し戻す。
Methodappend_row Sub append_row(ByVal values As Variant)1 行追加する

末尾に 1 行ぶんの配列を追加します。

入力values の一次元配列
出力なし。内部状態を更新。
前提配列長が列数と一致すること。
注意列数不一致は例外になります。
ユースケース集計行や追加入力行を後ろに付ける。
Methodappend_column Sub append_column(ByVal values As Variant)1 列追加する

末尾に 1 列追加します。空 Matrix に対しては列数だけを増やします。

入力values の一次元配列
出力なし。内部状態を更新。
前提行数がある場合は配列長が行数と一致すること。
注意空 Matrix の場合、値配列は不要です。
ユースケースフラグ列や計算結果列を追加する。

Transform

変形系

Methodtranspose Function transpose() As Matrix行列を転置する

行と列を入れ替えた新しい Matrix を返します。

入力なし
出力Matrix
前提読込済みで行数 1 以上であること。
注意行数 0 の Matrix はサポートしていません。
ユースケース縦持ちの一覧を横持ちの表へ変換する。

Output

出力系

Methodto_range Sub to_range(ByVal topLeft As Range)シートへ出力する

現在の Matrix をそのままシートへ書き戻します。

入力topLeft As Range
出力なし。シートへ書込。
前提読込済みで、行数 1 以上、topLeft が有効であること。
注意空 Matrix は出力できず例外になります。
ユースケース加工後の中間表を確認用シートへ貼り出す。

State

状態管理系

Methodclear Sub clear()内部状態を初期化する

保持している行列データと件数情報を初期状態へ戻します。

入力なし
出力なし。内部状態を初期化。
前提なし
注意読込済みフラグも False に戻ります。
ユースケース同じ変数で別の表を読み込む前にリセットする。
