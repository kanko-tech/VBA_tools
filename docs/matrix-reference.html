Matrix

矩形データの読み込み、行抽出、列選択、転置のための 2 次元クラスモジュール

  • 導入方法
  • 使い方
  • リファレンス

Properties

プロパティ

名前 型 説明
is_loaded Boolean 内部に矩形データが読み込まれているとき True を返します。
row_count Long データ行数を返します。
col_count Long データ列数を返します。
data Variant 内部で保持している 2 次元配列を返します。
item(rowIndex, colIndex) Variant 指定した行と列の交点にある値を返します。添字は 1 始まりです。

Methods

メソッド

カテゴリ シグネチャ 戻り値 説明
読込 read_range(rng As Range) Sub 矩形の Range を 2 次元配列として読み込みます。
読込 read_array(arr As Variant) Sub 2 次元配列をそのまま内部データとして読み込みます。
読込 read_empty(colCount As Long) Sub 指定列数を持つ空の Matrix を初期化します。
参照 row_values(rowIndex As Long) Variant 指定行を 1 次元配列として返します。
参照 col_values(colIndex As Long) Variant 指定列を 1 次元配列として返します。
抽出 filter_rows(mask As Variant) Matrix 真偽値マスクに合う行だけを持つ新しい Matrix を返します。
抽出 slice_rows(rowIndexes As Variant) Matrix 指定した行番号だけを取り出した新しい Matrix を返します。
抽出 select_columns(columnIndexes As Variant) Matrix 指定した列番号だけを残した新しい Matrix を返します。
更新 set_column_values(colIndex As Long, values As Variant) Sub 指定列の値をまとめて上書きします。
更新 append_row(values As Variant) Sub 末尾に 1 行追加します。
更新 append_column(values As Variant) Sub 右端に 1 列追加します。
形状変更 transpose() Matrix 行列を転置した新しい Matrix を返します。
出力 to_range(topLeft As Range) Sub 内部データを指定セルから矩形で書き戻します。
状態管理 clear() Sub 内部データと読み込み状態を初期化します。
