Example 1

案件数が 5 件以上の行だけ残す

Dim mat As New Matrix
Dim caseCounts As Variant
Dim mask() As Boolean
Dim i As Long

With Worksheets("貼付_部門別集計")
    mat.read_range .Range("B5:F10")
End With

caseCounts = mat.col_values(3)
ReDim mask(1 To mat.row_count)

For i = 1 To mat.row_count
    mask(i) = CLng(caseCounts(i)) >= 5
Next i

mat.filter_rows(mask).to_range Worksheets("貼付_部門別集計").Range("H5")

貼付済みの集計表から、レビュー対象の行だけを別ブロックへ抜き出す例です。B5:F10 はヘッダーを除いたデータ本体として扱っています。

Example 2

報告に必要な列だけを抜き出す

Dim mat As New Matrix

With Worksheets("貼付_部門別集計")
    mat.read_range .Range("B4:F10")
End With

mat.select_columns(Array(1, 4, 5)).to_range Worksheets("貼付_部門別集計").Range("H4")

もとの貼付表から「拠点」「4月売上」「5月売上」だけを抜き出し、比較用の簡易表を横に作る想定です。

Example 3

転置して月次比較用の見え方に変える

Dim mat As New Matrix

With Worksheets("貼付_部門別集計")
    mat.read_range .Range("B4:F10")
End With

mat.select_columns(Array(1, 4, 5)).transpose.to_range Worksheets("貼付_部門別集計").Range("N4")

横方向に並べた比較表が必要なときは、必要列だけを抜いたあとに転置すると整理しやすくなります。
