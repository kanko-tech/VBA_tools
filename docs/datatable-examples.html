Example 1

請求済の案件だけを抽出する

Dim tbl As New DataTable
Dim billedRows As DataTable

With Worksheets("売上明細")
    tbl.read_range .Range("B4:H12"), hasHeader:=True
End With

Set billedRows = tbl.filter_by_equals("請求状況", "請求済")
billedRows.to_range Worksheets("売上明細").Range("J4"), includeHeader:=True

ヘッダー付きの一覧から請求済だけを抜き出し、確認用の別表を同じシート右側に作る例です。

Example 2

優先区分の列を追加して埋める

Dim tbl As New DataTable
Dim amountVec As Vector
Dim highAmountMask As Variant

With Worksheets("売上明細")
    tbl.read_range .Range("B4:H12"), hasHeader:=True
End With

tbl.add_column "優先区分"
Set amountVec = tbl.col_vector("売上金額")
amountVec.fill_empty 0
amountVec.cast_to_double_safe emptyAsZero:=True, invalidAsZero:=True
highAmountMask = amountVec.gt(100000)

tbl.set_by_mask highAmountMask, "優先区分", "要確認"
tbl.set_by_equals "請求状況", "請求保留", "優先区分", "確認待ち"
tbl.to_range Worksheets("売上明細").Range("J4"), includeHeader:=True

元の列を壊さずに派生列を足し、売上金額や請求状況をもとに優先度を付ける実務寄りの例です。

Example 3

拠点を絞って売上順に並べ替える

Dim tbl As New DataTable
Dim picked As DataTable

With Worksheets("売上明細")
    tbl.read_range .Range("B4:H12"), hasHeader:=True
End With

Set picked = tbl.filter_by_in("拠点", Array("東京", "大阪"))
picked.sort_by "売上金額", ascending:=False
Set picked = picked.select_columns(Array("受注ID", "拠点", "担当者", "売上金額", "請求状況"))
picked.to_range Worksheets("売上明細").Range("J4"), includeHeader:=True

列名ベースで絞り込み、必要列だけに整えたうえで売上順に並べ替えることで、共有しやすい一覧を短く書けます。
